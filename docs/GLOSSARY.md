# Glossary

- Adapter or interface: a Windows network connection such as Wi-Fi, Ethernet, or VPN.
- Auto DNS: Windows obtains DNS servers automatically, usually from DHCP or network policy.
- DDNS: Dynamic DNS, a service that updates a DNS record when your public IP changes.
- DNS: Domain Name System, the service that turns names such as `example.com` into IP addresses.
- DNS profile: a named list of DNS server IP addresses saved in the app.
- Gateway: the local router or next hop used to reach external networks.
- HKCU: HKEY_CURRENT_USER, the Windows registry hive for the current user.
- PAC URL: Proxy Auto-Config URL, a script address Windows can use for automatic proxy rules.
- Plugin: a trusted local Python extension loaded by the app.
- Proxy bypass: host patterns that should skip the configured proxy.
- Public IP: the external IP address seen by internet services.
- Restore point: the most recent app-captured DNS/proxy state used by Restore.
- WinINet: Windows current-user internet settings used by many desktop apps and browsers.
